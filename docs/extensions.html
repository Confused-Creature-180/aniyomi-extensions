How to Install

After downloading an APK, you can install it directly or add the repo to Aniyomi for automatic updates.

1

Direct APK Install

Download the APK and install manually.

  1. Tap Download APK on any extension card above.
  2. Open the downloaded file on your Android phone.
  3. Allow installation from unknown sources if prompted.
  4. Open Aniyomi — the extension appears in Extensions.
2

Add Repo for Updates

Get automatic updates by adding the repo.

  1. Tap Add Repo to Aniyomi above.
  2. Confirm in Aniyomi when prompted.
  3. Install extensions from within Aniyomi.
  4. Updates arrive automatically when new versions are published.
