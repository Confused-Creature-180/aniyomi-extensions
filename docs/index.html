Quick Install

Two ways to add this repository to Aniyomi on your Android device.

1

One-Tap Install

Quickest method — opens Aniyomi directly.

  1. Tap the "Add to Aniyomi" button above on your Android phone.
  2. Aniyomi opens and prompts you to confirm adding the repository.
  3. Tap Confirm. The repo is now added.
  4. Go to Extensions, find your extension, tap Install.
Add to the app
2

Manual Install

If the one-tap link doesn't work on your browser.

  1. Copy the repo URL (tap Copy URL above).
  2. Open Aniyomi on your phone.
  3. Go to Settings → Extensions (puzzle icon).
  4. Tap ≡ menu → Manage extension repos → +.
  5. Paste the URL, confirm, and install extensions.
