How to Install

Two ways to add this repository to Aniyomi on your Android device.

1

One-Tap Install

Quickest method — opens Aniyomi directly.

  1. Tap the "Add to Aniyomi" button at the top of this page on your Android phone.
  2. Aniyomi opens and prompts you to confirm adding the repository.
  3. Tap Confirm. The repo is now added.
  4. Go to Extensions → find your extension under its language → tap Install.
Add to Aniyomi
2

Manual Install

If the one-tap link doesn't work on your browser.

  1. Copy the repo URL (tap Copy URL at the top).
  2. Open Aniyomi on your phone.
  3. Go to Settings → Extensions (puzzle icon).
  4. Tap the ≡ menu → Manage extension repos → + (Add repo).
  5. Paste the URL → Confirm.
  6. Find the extension under its language section → tap Install.
