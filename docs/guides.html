Guides & Tutorials

Everything you need to know — from installing Aniyomi to troubleshooting extension issues. Tap any guide to expand it.

Install Aniyomi

Get the Aniyomi app on your Android device

Step 1: Download Aniyomi

Aniyomi is available from the official GitHub releases page. Do not download from third-party sources.

  1. Go to github.com/aniyomiorg/aniyomi/releases
  2. Find the latest stable release.
  3. Download the aniyomi-vN.N.N.apk file to your Android phone.

Step 2: Enable Unknown Sources

Android requires you to allow installation from unknown sources:

  • Go to Settings → Apps → Special Access → Install Unknown Apps
  • Select your browser and enable "Allow from this source".

Step 3: Install

Tap the downloaded APK file and confirm the installation. Open Aniyomi when done.

Add an Extension Repository

Add this (or any) repo to Aniyomi so you can browse and install extensions

Method 1: One-Tap Deep Link

The easiest method — works on Android when tapping from your phone's browser:

  1. Go to our Add Repo page on your phone.
  2. Aniyomi opens automatically and prompts you to add the repo.
  3. Tap Confirm.

Method 2: Manual URL

If the deep link doesn't work, add the repo manually:

  1. Copy this URL:
https://raw.githubusercontent.com/Confused-Creature-180/aniyomi-extensions/repo/index.min.json
  1. Open Aniyomi.
  2. Go to Settings → Extensions (the puzzle icon).
  3. Tap the ≡ menu (top-right) → Manage extension repos.
  4. Tap the + button.
  5. Paste the URL and tap OK / Add.

The repo now appears in your extension repos list. Its extensions show up under their language sections in the Extensions page.

Install an Extension

Install individual extensions from a repo you've added

From within Aniyomi

  1. Open Aniyomi.
  2. Go to Extensions (the puzzle icon).
  3. Scroll to the language section of the extension you want (e.g. "English").
  4. Find the extension and tap Install.
  5. Wait for the download and installation to complete.

From the Website

  1. Go to our Extensions page.
  2. Find the extension you want.
  3. Tap Download APK to get the file directly.
  4. Open the downloaded APK on your phone to install it.

Note: Installing via the repo (first method) is recommended — you'll get automatic updates. Direct APK installs won't auto-update unless you also add the repo.

Update Extensions

How updates work and how to manually check for them

Automatic Updates

Aniyomi checks for extension updates automatically when you open the app. If an update is available, an Update button appears next to the extension in the Extensions page.

Manual Check

  1. Open Aniyomi → Extensions.
  2. Tap the ≡ menu → Update extensions.
  3. Aniyomi fetches the latest index.min.json and compares version codes.

How Updates Are Detected

Each extension has a versionCode (a number). When the repo's version code is higher than your installed one, Aniyomi offers an update. The APK is signed with the same keystore, so the update installs seamlessly — your saved anime, history, and settings are preserved.

Troubleshooting

Common issues and how to fix them

"Repository not found" / Invalid URL

Make sure the URL ends with /index.min.json (not /index.json). Copy the exact URL from our home page.

"Signature mismatch" on update

This happens if you previously installed the same extension from a different repo (with a different signing key). Uninstall the old extension first, then install from our repo.

Extension doesn't load / "Source error"

  • The scraped website may be down or changed its structure. Report it on GitHub.
  • Check your internet connection.
  • Try clearing Aniyomi's cache (Settings → Advanced → Clear cache).

Extension not showing in the list

  1. Make sure the repo was added successfully (check Settings → Extensions → Repos).
  2. Pull down to refresh the extensions list.
  3. Look under the correct language section.
  4. If it's an NSFW extension, enable NSFW sources in Settings.

App crashes after installing

Report the crash on GitHub Issues with your Android version, Aniyomi version, and the extension name. Include the crash log if possible.

FAQ

Frequently asked questions

Is this affiliated with Aniyomi?

No. This is an independent third-party repository. We are not affiliated with the Aniyomi development team. For app issues, use the official Aniyomi support. For extension issues, use our GitHub Issues.

Are these extensions free?

Yes, completely free and open source under the Apache 2.0 license.

Do you host the anime content?

No. We only host the extension plugins. The extensions allow Aniyomi to interface with third-party streaming websites. We don't host, stream, or store any video content.

How often are extensions updated?

When a scraped website changes its structure and breaks an extension, we try to fix and push an update as soon as possible. There's no fixed schedule.

Can I request a new extension?

Yes! Open an issue on GitHub with the site URL and details. Requests are handled on a best-effort basis.

Why does the extension name show "Aniyomi: ..."?

That's the Aniyomi convention — every extension's app label is prefixed with "Aniyomi: ". This is set by the Aniyomi framework, not by us. The actual source name (what appears in the Aniyomi source list) is set by the extension.
