## Quick Start

### Add the repo to Aniyomi

**Option 1 — One Tap:** Tap the button above on your Android phone (opens Aniyomi directly).

**Option 2 — Manual:**
1. Open **Aniyomi** → **Settings → Extensions** (puzzle icon)
2. Tap **≡ → Manage extension repos → +** (Add repo)
3. Paste this URL:

```
https://raw.githubusercontent.com/Confused-Creature-180/aniyomi-extensions/repo/index.min.json
```

4. **Confirm** → browse extensions under their language section → **Install**.

### Need help?

Visit the **[website](https://confused-creature-180.github.io/aniyomi-extensions/)** for a full guide, extension list, and visual walkthrough.
