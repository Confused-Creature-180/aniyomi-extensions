# Aniyomi Anime Extensions

A custom extension repository for [Aniyomi](https://github.com/aniyomiorg/aniyomi), the free and open-source anime streaming app for Android.

## Available Extensions

| Extension | Language | Version | NSFW |
|-----------|----------|---------|------|
| AniKoto 180 | English (en) | 16.5 (code 5) | No |

## How to Add This Repo to Aniyomi

### Option 1: One-Tap Deep Link

Tap this link on your Android phone (opens Aniyomi directly):

```
aniyomi://add-repo?url=https%3A%2F%2Fraw.githubusercontent.com%2FConfused-Creature-180%2Faniyomi-extensions%2Frepo%2Findex.min.json
```

### Option 2: Manual URL

1. Open **Aniyomi** on your phone
2. Go to **Settings** → **Extensions** (puzzle icon)
3. Tap the **menu (≡)** → **Manage extension repos** → **+ (Add repo)**
4. Paste this URL:

```
https://raw.githubusercontent.com/Confused-Creature-180/aniyomi-extensions/repo/index.min.json
```

5. Confirm. The extensions listed above will appear under their language section.
6. Tap **Install** on the extension you want.

## Updating Extensions

Aniyomi checks for updates automatically. When a new version is published to this repo, you'll see an **Update** button next to the extension in Settings → Extensions.
