## Quick Start

### Add the repo to Aniyomi

**Option 1 — One Tap:** Tap the "Add to Aniyomi" badge above **on your Android phone**. Aniyomi opens and prompts you to add the repo.

**Option 2 — Manual:**

1. Open **Aniyomi** on your phone
2. Go to **Settings → Extensions** (puzzle icon)
3. Tap **≡ → Manage extension repos → +** (Add repo)
4. Paste this URL:

```
https://raw.githubusercontent.com/Confused-Creature-180/aniyomi-extensions/repo/index.min.json
```

5. **Confirm** → browse extensions under their language section → **Install**

### Need help?

Visit the **[website](https://confused-creature-180.github.io/aniyomi-extensions/)** for a full extension list with download buttons, step-by-step guides, and a detailed disclaimer.

## Repository Structure

| Branch | Purpose |
|--------|---------|
| `main` | Source code, README, and the GitHub Pages site (in `/docs`) |
| `repo` | Published extensions — `index.min.json`, `index.json`, `repo.json`, `apk/`, `icon/` |

> The `repo` branch is auto-generated and contains **only** published artifacts (no source code). It uses orphan commits (separate history from `main`). If GitHub shows a "Compare & pull request" banner for the `repo` branch, **dismiss it** — it is not a PR target.
